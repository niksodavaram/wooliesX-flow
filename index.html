Edge-to-cloud flow for a single POS event
WooliesX stack visualised from in-store hardware up to AI and customer experience.
Prototype created by Nireekshan Sodavaram for WooliesX interview on 7 April 2026.

This is a narrative map of a single self-checkout scan across 9 layers of the WooliesX stack – with my domain highlighted.

Use the numbered dots, Back/Next buttons, or keyboard (← / → / Home / End) to walk the flow. Green-highlighted pills show technologies I directly own.

Legend: green outline = layers in my direct domain; green tech pills = technologies I own or actively work with. This prototype is implemented with vanilla HTML/CSS/JS and designed to run on a lightweight static host (Vercel).
